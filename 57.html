Задачи на продвинутую работу с контекстом в JavaScript

Работа с call

Задача 1. Пусть дан следующий код: var elem = document.getElementById('elem'); function func() { alert(this.value); } func(); //тут должно вывести value инпута Добавьте в последнюю строчку метод call() так, чтобы на экран вывелось value инпута, лежащего в переменной elem.

Задача 2. Пусть дан следующий код: var elem = document.getElementById('elem'); function func(surname, name) { alert(this.value + ', ' + surname + ' ' + name); } func(); //тут должно вывести 'привет, Иванов Иван' Добавьте в последнюю строчку метод call() так, чтобы на экран вывелось 'привет, Иванов Иван'. Слово 'привет' должно взяться из value инпута, а 'Иванов' и 'Иван' должны быть параметрами функциями.
