События Drag-and-Drop

Перетягиваемый элемент

Задача 1. Дан див для перетягивания. Сделайте так, чтобы в момент начала перетягивания его граница красилась в красный цвет.

Задача 2. Дан див для перетягивания. Сделайте так, чтобы в момент окончания перетягивания его граница красилась в зеленый цвет.

Задача 3. Дан див для перетягивания. Сделайте так, чтобы в процессе перетягивания в консоль выводился '!' (каждый раз при срабатывании события).

Принимающий элемент

Задача 4. Дан элемент для перетягивания и принимающий див. Сделайте так, чтобы по дропу граница принимающего дива красилась в красный цвет.












Задача 5. Дан элемент для перетягивания и принимающий див. Сделайте так, чтобы при движении элемента над принимающем дивом граница принимающего дива в виде точечек.












Задача 6. Дан элемент для перетягивания и принимающий див. Сделайте так, чтобы при вхождении элемента в границы принимающего дива в текст принимающего дива записывался '!', а при уходе - этот текст убирался.
